## Project Context

- **Project:** confirmation-saints
- **User:** Jorge Balderas
- **Created:** 2026-04-12
- **Language:** Swift / SwiftUI
- **Platform:** iOS (iPhone + iPad), future Android
- **Description:** An iOS app helping Catholic confirmation candidates find and choose a patron saint. Primary audience is teenagers, also supports adults receiving the sacrament later. Features saint search by name, patron day, affinity, country, age, married status, etc. Multilingual (English default, Spanish). Content sourced from Loyola Press, Focus, Lifeteen, Ascension Press, Hallow, etc. with attribution.
- **Key Requirements:**
  - Self-contained with easy content updates
  - Explain confirmation and saint-choosing tradition
  - Multi-lingual (EN default, ES)
  - Cross-platform ready (iOS first, Android later — reuse content)
  - Include saint images with attribution
